# Social Media Engagement Analysis
## Project Overview
This project analyzes social media engagement data using Python.

The dataset contains:

- User demographics
- Post types
- Likes, comments, shares
- Engagement metrics
- Sentiment analysis
- Hashtag analysis
## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
## Key Analysis Performed
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Statistical Analysis
- Visualization
## Insights
- Video posts received high average likes.
- India had the highest average impression count.
- Engagement rate had negative correlation with impression count.
- Most users used 1–3 hashtags.
